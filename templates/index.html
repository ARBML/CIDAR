Arabic Alpaca Annotation Tool

The Alpaca translation tool aims to create an Arabic version of the Alpaca dataset. We initially translted 52k samples using ChatGPT but around 10K of the samples have some issues. Here is a summary of some exmaples:

  1. Fix Grammatical Issues
    Some words might not be translated correctly especially at the start, we want all the statements to start by a question For example تحديد should be replaced by حدد or قم بتحديد
  2. Fix Translation Issues
    Some inputs or outputs might not be translted. Also, some instructions might be specific to English please give their corropsonding examples in Arabic.
  3. Fix Origianl Alpaca
    Some examples in the original English Alpaca might not be correct. For example some insturctions might have some missing inputs.
